Wordle Remix

About

This is a remix to the classic Wordle game. It is currently very similar to the original game. To play, just start typing! Press enter when you want to submit your guess.

If a letter looks like this, it means it is in the correct location.

p

If a letter looks this this, it means it is in the word but in the incorrect location.

n

If a letter remains grey like below, it is not in the word.

b
